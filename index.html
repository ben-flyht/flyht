What's so good about Ruby on Rails?

Great question; glad you asked. Yes, some people write code with PHP, Python or Perl but there are some distinguishing factors about Ruby (the language) and Rails (the framework) that in our experience make them the tools of choice when building innovation projects. They allow us to write concise, human-centric and dare we say it, beautiful code. We're able to bring your ideas to life elegantly and quickly and that makes us (and hopefully you) very happy. If you wanted to dig a little deeper, maybe have a read about the Rails doctrine.

Can we host elsewhere?

Yes you can. Truth be told, hosting is the least exciting part of what we do. All the code we write is contractually owned by you. If you want to host it somewhere else, you are absolutely free to. We care about speed and efficiency and that's why we're pretty selective with who we use ourselves. Heroku is built on top of Amazon Web Services so provides exceptional scale and performance and is designed to minimise development cost. If it puts your mind at rest, one of our clients regularly sees 10k requests per minute averaging a page response time under 60ms which (if you care about this stuff) is very good.
